# Meet

### This app is for video call for Abiteck company
### Used technologies:
#### Vite, Vue, Vuex, TypeScript, JavaScript, Bootstrap, Socket.io, WebRTC

### npm install:
#### *npm install*


### __get up server:__
#### *npm start:dev-server*

### __get up frontend:__
#### *npm run server*

### __get build:__
#### *npm run build*
